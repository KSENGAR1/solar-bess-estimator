# Solar + BESS Financial Estimator

A professional web-based financial analytics dashboard for utility-scale Solar + Battery Energy Storage System (BESS) projects.

This project was developed to estimate:
- Solar plant CAPEX
- Battery Energy Storage System (BESS) costs
- Annual energy generation
- Revenue forecasting
- O&M expenses
- Annual profit
- Simple payback period

The estimator provides an interactive and responsive UI for modeling renewable energy project economics in real time.

# Financial Model Logic

The application estimates project economics using industry-style assumptions.

## Annual Energy Generation

```math
Energy = MW × 8760 × CUF
```

## Annual Revenue

```math
Revenue = Energy × Tariff
```

## O&M Cost

```math
O&M = 2% × Total CAPEX
```

## Simple Payback

```math
Payback = Total CAPEX / Annual Profit
```

# Installation & Setup

Clone the repository:

```bash
git clone https://github.com/KSENGAR1/solar-bess-estimator.git
```

Move into project directory:

```bash
cd solar-bess-estimator
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```
